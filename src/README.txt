Paginebianche.it è una fonte di dati inesauribile per commerciali e promotori, però utilizzare 
l'interfaccia web può essere molto lento se i dati da cercare sono tanti. Perciò ho realizzato
un programma in Java che automatizza la ricerca di molti nomi.

Pagine biache cambia, perciò puoi cercare la versione più recente a questo indirizzo 

http://digilander.libero.it/caiofior/paginebianche_data_extractor.html

O modificare il programma, tieni presente che nel file config.ini trovi l'url di pagine bianche
e le espressioni regoalri utilizzate per estrarre i dati
